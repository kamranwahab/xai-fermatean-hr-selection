## 📌 Overview
This repository contains the official Python implementation, empirical validation matrices, and robustness audit suite for our research manuscript (targeted for *Expert Systems with Applications*). 

The codebase introduces a 4-layer decision-support architecture designed to resolve the tension between predictive machine learning efficiency and algorithmic transparency in human resources. 

### Core Pipeline Components:
1. **Unsupervised Target Synthesis:** $k$-means clustering to eliminate human labeling bias in historical Likert data.
2. **Predictive XAI Layer:** XGBoost classification paired with SHapley Additive exPlanations (SHAP) for monotonic, legally auditable feature attribution.
3. **Ontological Hesitation Bridging:** Dynamic calculation of Fermatean hesitation ($\pi$) using TF-IDF cosine similarity and Breadth-First Search across the European Union’s ESCO occupational graph.
4. **Fuzzy MCDM Engine:** Subjective expert weighting via the Full Consistency Method (FUCOM), integrated with a novel Fermatean Fuzzy Einstein-Weighted MARCOS ranking algorithm.
